myFlix: a REST API with node.js

This is the server-side component of a “movies” web application. This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies

This allows new users to register /users
http://localhost:8080/users
 POST TBD Successful POST request adding a new user
This allows users to update their info/username /users/info
http://localhost:8080/users/info
 PUT TBD Successful PUT request updating info on a specific user
This allows users to add a movie to their favorites list /users/movies/favorites/:Title
http://localhost:8080/users/movies/favorites/RBG
 POST None Successful POST request adding a specific movie to a favorites list of a specific user
This allows users to remove movies from favorites /users/movies/favorites/:Title
http://localhost:8080/users/movies/favorites/RBG
 DELETE None Successful DELETE request deleting specified title from existing user
This allows users to deactivate their account /users
