myFlix: a REST API with node.js

This is the server-side component of a “movies” web application. This web application provides users with access to information about different movies, directors, and genres. Users can sign up, update their personal information, and create a list of their favorite movies!

This endpoint will get the description about a single movie title /movies/:Title

"https://jennysflix.herokuapp.com/movies/Miss%20Congeniality"
 GET None Description of the specified movie title.

Example response: 


"An F.B.I. Agent must go undercover in the
Miss United States beauty pageant to prevent

Return data about the genre of a single movie by title /movies/genres/:Name

"https://jennysflix.herokuapp.com/movies/genres/Comedy"
GET None Description of the specified genre.

Example response: 


"Comedy is a genre of film in which the main emphasis is on
 humor. These films are designed to make the audience laugh
 through amusement and most often work by exaggerating
 characteristics for humorous effect."
This will return data about a movie director by name /movies/directors/:Name

"https://jennysflix.herokuapp.com/movies/directors/Julie%20Cohen"
GET None A JSON object with Name, Bio, and Birthday of the specified director.

Example response: 


{
    "Name": "Julie Cohen",
    "Bio": "a producer and director, known for RBG (2018)",
    "Birth": "1972"
}
This allows new users to register /users

"https://jennysflix.herokuapp.com/users"
 POST A JSON object with new user data.
Example of request body: 

{
  Username: "LeslieKnope",
  Password: "jellybeans123",
  Email: "jennyP@gmail.com",
  Birthday: "10/31/1996"
}
A JSON object with data about the new user in this format: 

{
  ID: Integer,
  Username: String,
  Password: String,
  Email: String,
  Birthday: Date
}

Example response:

{
    "FavoriteMovies": [],
    "_id": "60664a6dbe02655f8423ab56",
    "Username": "LeslieKnope",
    "Password": "jellybeans123",
    "Email": "jennyP@gmail.com",
    "Birthday": "1996-10-31T07:00:00.000Z",
    "__v": 0
}
This allows users to update their info/username /users/:Username

"https://jennysflix.herokuapp.com/users/LeslieKnope"
PUT A JSON object with updated user data.
Example of request body: 

{
  "Username": "LeslieKnope1",
  "Password": "newPassword789",
  "Email": "LeslieKnope@gmail.com",
  "Birthday": "10/31/1996"
}
A JSON object with data about the updated user in this format: 

{
  Username: String,(required)
  Password: String,(required)
  Email: String,(required)
  Birthday: Date
}

Example response:

{
    "FavoriteMovies": [],
    "_id": "60664a6dbe02655f8423ab56",
    "Username": "LeslieKnope1",
    "Password": "newPassword789",
    "Email": "LeslieKnope@gmail.com",
    "Birthday": "1996-10-31T07:00:00.000Z",
    "__v": 0
}
This allows users to add a movie to their FavoriteMovies list /users/:Username/movies/:MovieID

"https://jennysflix.herokuapp.com/users/LeslieKnope1/Movies/605cf6e67759ae52096e6f03"
POST None A JSON object of the user's data, including the updated FavoriteMovies list.

Example response: 


{
    "FavoriteMovies": [
        "605cf6e67759ae52096e6f03"
    ],
    "_id": "60664a6dbe02655f8423ab56",
    "Username": "LeslieKnope1",
    "Password": "newPassword789",
    "Email": "LeslieKnope@gmail.com",
    "Birthday": "1996-10-31T07:00:00.000Z",
    "__v": 0
}
This allows users to remove movies from their favorite movies array /users/:Username/favoritemovies/:MovieID

"https://jennysflix.herokuapp.com/users/LeslieKnope/favoritemovies/605cf6e67759ae52096e6f03"
DELETE None Confirmation that specified movie has been removed from the user's favorite movies.

Example response: 


Movie ID 605cf6e67759ae52096e6f03 was deleted from user LeslieKnope
This allows users to deactivate their account /users

"https://jennysflix.herokuapp.com/users/LeslieKnope"
 DELETE None Example response:

User LeslieKnope was deleted.
